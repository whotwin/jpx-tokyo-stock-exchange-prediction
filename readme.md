股票价值预估，原赛事的要求是需将这些股票按预期收益率从高到低排序，评审标准将是排名前200的股票与排名后200的股票之间的收益率差异。
那么这个预测最好提前多久可能会是可靠的，提前1天、1周还是1个月？如果预测的是1个月后的股票价值、收益率的话，那么这一个月中是不是就不能换仓或平仓。
现在的问题是，预测1天后的收益可能噪声过大，预测1个月后的数据并且中途不允许其他操作的话可能会忽略这一个月期间的部分突发情况导致的意外变化，如果预测1个月后的数据并允许进行一些中途的调整可能比较合理但是任务会复杂很多。
像原赛事那样固定预测前200名的股票和后200名的股票比较合理，还是直接预测比如2000支股票中每支股票在未来x天时的收益比较合理。
目前模拟真实情况，为每支股票赋予了权重，对股票的买进卖出加权求和得到x天后的收益。如果简化问题的话可以只预测1天后的收益排名，而不考虑要进行的操作。
如果上面的说法都不对的话，那么现实里一般是怎么个流程。
